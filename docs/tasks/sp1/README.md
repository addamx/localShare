# SP1 任务拆解

## 任务说明

本阶段对应 Tauri 云剪贴板 MVP 的首个实现周期，目标是完成“桌面监听纯文本剪贴板 + 手机局域网访问与提交 + 历史管理基础能力”的可交付闭环。

任务拆解依据：

- [产品需求文档](../../demand/产品需求文档.md)
- [技术设计文档](../../design/tauri-cloud-clipboard-design.md)

共享约束统一沉淀在 `shared` 目录，具体任务只引用，不重复展开：

- [shared/范围与验收基线.md](/home/addams/projects/localShare/docs/tasks/sp1/shared/范围与验收基线.md)
- [shared/实现约束与公共约定.md](/home/addams/projects/localShare/docs/tasks/sp1/shared/实现约束与公共约定.md)

## 任务清单

| 编号 | 任务 | 目标 |
| --- | --- | --- |
| T01 | [项目骨架与运行基线](./T01-项目骨架与运行基线.md) | 建立 Tauri 2、Vue 3、Rust、SQLite、配置与目录基线 |
| T02 | [数据层与持久化基线](./T02-数据层与持久化基线.md) | 完成 SQLite schema、迁移、仓储接口与数据规则 |
| T03 | [剪贴板监听与事件链路](./T03-剪贴板监听与事件链路.md) | 完成桌面剪贴板轮询、去重入库、当前项更新和事件广播 |
| T04 | [会话认证与局域网服务](./T04-会话认证与局域网服务.md) | 完成 token、鉴权、REST API、SSE、健康检查与服务状态 |
| T05 | [桌面端状态台与历史管理 UI](./T05-桌面端状态台与历史管理UI.md) | 完成桌面端三栏或双栏主界面及历史管理操作 |
| T06 | [移动端 H5 与手机操作闭环](./T06-移动端H5与手机操作闭环.md) | 完成手机页面、历史浏览、文本提交和激活操作 |
| T07 | [联调、异常反馈与发布前收口](./T07-联调异常反馈与发布前收口.md) | 完成跨端联调、失败提示、端口与权限异常处理、验收收口 |

## 推荐执行顺序

1. T01 项目骨架与运行基线
2. T02 数据层与持久化基线

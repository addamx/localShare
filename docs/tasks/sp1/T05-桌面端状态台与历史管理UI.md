## 任务目标

实现桌面端主控制台，覆盖状态展示、二维码入口、历史列表、详情面板和主要历史操作，形成本地可用的主工作台。

## 任务范围

- 顶部状态栏展示设备名、局域网地址、服务状态、连接人数、token 过期时间
- 左侧历史列表展示记录、搜索、置顶、删除、清空、当前项标记
- 右侧详情展示完整文本、来源、时间、字符数和激活操作
- 右上角二维码卡片展示访问入口与刷新 token 操作
- 处理服务不可用、端口冲突、权限不足等错误提示

## 产出物

- 桌面端主界面
- 历史列表与详情组件
- 状态展示与二维码卡片
- 桌面端命令调用与事件刷新逻辑

## 完成标准

- 桌面端可完成历史查看、搜索、置顶、删除、清空和激活
- 新内容进入时列表能及时刷新并有明确当前项
- 服务状态和二维码信息始终可见
- 失败状态具备明确提示，不是静默失败

## 实施提示

- 搜索只影响视图过滤，不影响底层写入链路
- 置顶排序要和时间排序规则兼容
- 二维码卡片应和 token 刷新结果同步更新

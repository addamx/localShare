## 任务目标

实现局域网 HTTP 服务、token 会话、鉴权与 REST/SSE 接口，支撑手机端访问、桌面端状态展示和跨端写入闭环。

## 任务范围

- 建立 LAN HTTP Server，提供 H5 静态资源与 API
- 实现 `GET /api/v1/session`、`POST /api/v1/session/rotate-token`、`GET /api/v1/health`
- 实现历史列表、详情、提交、激活、置顶、删除、SSE 等接口
- 实现 Bearer token 校验、二维码 URL token 传递、过期与轮换
- 实现请求日志、基础限流、错误码和统一返回格式

## 产出物

- HTTP 路由与 handler
- Session/Auth 模块
- REST API 与 SSE 订阅能力
- 服务状态与端口绑定逻辑

## 完成标准

- 手机端可通过局域网 URL 访问服务
- 写接口全部受鉴权保护
- token 过期后旧入口失效
- 统一错误码和响应结构可被前端稳定消费
- 健康检查、端口异常和服务状态可对外暴露

## 实施提示

- 端口冲突要有自动回退或明确错误信息
- 读写接口的鉴权和限流逻辑要尽量统一在中间层处理
- SSE 断连后不要求复杂重放，但需要支持刷新恢复
